This is a project that was for the Spring 2025 semester at Blackhawk Technical College. It is supposed to be a simple application that would hold simple student records. This project is not something that is to be used in production code in current form due to it not containing a full user system. That is, the "login" just consists of checking a hard coded password that should not be used in the real world.

To see this project, it is best to use XAMPP and run the `[database.sql](./database.sql)` script.
