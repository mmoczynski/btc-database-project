This was a project that was for the Spring 2025 semester at Blackhawk Technical College. It was supposed to be a simple application that would hold simple student records. This project is not something that is to be used in production code in current form due to it not containing a full user system. That is, the "login" just consists of checking a simple hard coded password. This behavior can be changed by modifying the `login` function found in [./includes/prepend.php](./includes/prepend.php).

To utlize this project, it is best to use XAMPP and run the [database.sql](./database.sql) script in PHPMyAdmin. [You can also click here for screenshots](./docs/screenshots.md).
